紙とWebのデザインの違い
紙のデザインとWebのデザインでは基本的なデザインスキルは共通していますが、制作過程を含めて大きな違いがあります。
今回はその違いについて紹介していきます。
全体の方向性

紙

紙の種類、印刷方法、加工方法などを決める。

Web

見た目だけでなく、動き方やSEOも考える。

フォント

紙

自由

Web

環境によって制限される（ゴシック体が推奨されている）。WEBの場合10px以下はあまり使用しない。

no image
色

紙

CMYK（減法混合・減法混色） 誰が見ても同じ色を出すことができる。 色域が狭く、色が沈みがち。

Web

RGB（加法混合・加法混色） 見る環境によって色味が変化する（特にオレンジは差が出やすい） 色域が広く、鮮やかに表現できる。

no image
レイアウト

紙

最も伝えたい情報が飛び込んでくるようにする。 用紙サイズに合わせて自由に文字や画像を配置。 配置されたコンテンツは固定され、誰が見ても同じ構成。 左右に読み進める「Z型」のレイアウトが推奨されている。

Web

最も伝えたい情報をファーストビューに配置する。メディアによって見え方が変わるので 「可変」することを意識してレイアウトする。縦に読み進める「F型」のレイアウトが推奨されている。

no image
単位

紙

mmなどの絶対単位。

Web

pxなどの相対単位。（解像度や表示されるデバイスによって変化）

no image
校正

紙

納品すれば完了。修正は不可能。

Web

納品後も更新作業がある。

no image
ターゲット

紙

「どこで？」「いつ？」見られるのかが想像しやすく、 部数に限りがあるため、あらかじめターゲットを絞る。

Web

「どこでも」「いつでも」見られるため、ターゲットが絞りにくい。

no image


紙とWebの大きな違いは「メディアが固定されているか」「動きや操作性があるか」ですが、 根本的な部分は共通しています。Webデザインの際は「可変」することを意識し、柔軟性があるレイアウトを心がけていきましょう。
